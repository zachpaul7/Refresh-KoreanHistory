# 한국사 새로고침

## 프로젝트 소개
Unity 엔진을 기반으로 제작된 2D 횡스크롤 어드벤처 게임

SW중심대학 공동 HACKATHON 2023에서 제작하게 되었다. 
 - 최근 낮아진 사람들의 역사 의식을 게임을 통해 쉽게 재인식 시키는자는 기획으로 유니티 제작에 게임 로직(스크립트) 제작 담당으로 참여
 - 팀이름은 SW:ORD 로 프로젝트명은 한국사 새로고침

## 프로젝트 개요
- 개발기간: 3일
  
- 개발도구: Unity
  
- 개발언어: C#
  
- 플랫폼 : 안드로이드
